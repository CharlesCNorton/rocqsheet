# rocq-crane-imgui (skeleton)

Reusable Crane bindings for Dear ImGui, structured the same way as
`rocq-crane-sdl2` for the reference SDL2-based projects.  The current
contents of `theories/ImGuiE.v` and `src/imgui_helpers.h` in the
parent rocqsheet repository should migrate into this directory; once
that's done it will be pinned via `opam pin add rocq-crane-imgui
./rocq-crane-imgui` and the rocqsheet tree will consume it as a
submodule.

The actual repo split has not yet happened; this directory is a
placeholder reserving the layout.
